# Variety Of Milkshakes

A responsive card-based website showcasing different milkshake varieties with chef ratings and descriptions.

## Features

- **Responsive Card Layout** - Three centered cards displaying milkshake options
- **Image Hover Effect** - Images scale up smoothly on hover (1.05x)
- **Chef Ratings** - Each milkshake includes chef name and star ratings
- **Gradient Background** - Dark to light linear gradient background
- **Rounded Cards** - Modern card design with border-radius
- **Shadow Effects** - Box shadows for depth and visual appeal
- **Description Text** - Fixed height descriptions (80px) to maintain uniform card sizes

## Technologies Used

- HTML5
- CSS3 (Flexbox, Gradients, Transitions, Transforms)

## Key CSS Classes

| Class | Purpose |
|-------|---------|
| `.card_wrapper` | Flex container for centering cards |
| `.milkshake_card` | Individual card styling |
| `.milkshake_img` | Image container with shadow |
| `.container` | Text content wrapper |
| `.meta` | Chef info and ratings section |

## Usage

Simply open `index.html` in a web browser to view the milkshake cards.
